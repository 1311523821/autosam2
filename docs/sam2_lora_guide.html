🧮 损失函数可视化面板

Tversky Loss

Tversky Index
TI = TP / (TP + α·FN + β·FP)
α 控制漏检惩罚，β 控制误检惩罚
小目标配置
L = 1 - TIα=0.7, β=0.3
α > β：漏检比误检代价高 2.3×

Dice Loss

Dice / F1 Score
Dice = 2·TP / (2·TP + FN + FP)
Tversky 的特例 (α=β=0.5)
Dice Loss
Ldice = 1 - Dice
对称惩罚 FN 和 FP

Dice 对小目标不友好：小目标的 |pred| 和 |gt| 都很小，Dice 对边界像素的惩罚不足，容易产生模糊预测。

Focal Loss

Focal Loss
FL = -α(1-pt)γlog(pt)
γ 放大难样本权重，减轻类不平衡
组合损失
Ltotal = Ltversky + Lfocal
分割精度 + 难样本挖掘
