# 评估指标说明

本文档详细说明引导头训练使用的评估指标及其物理意义。

---

## Hit Rate@R（命中率）

### 定义

**Hit Rate@R** = 预测点与 GT 中心距离 ≤ R 像素的样本比例

```
Hit Rate@R = count(distance ≤ R) / total_samples
```

### 计算方式

1. 从预测热力图中提取最大值位置作为预测点 (pred_x, pred_y)
2. 计算 GT mask 的质心作为真值点 (gt_x, gt_y)
3. 计算欧氏距离：`distance = sqrt((pred_x - gt_x)² + (pred_y - gt_y)²)`
4. 统计距离 ≤ R 的样本比例

### 不同 R 值的物理意义

| R 值 | 物理意义 | 对 SAM 2 的影响 |
|------|----------|-----------------|
| **5px** | 高精度命中，几乎完美 | SAM 2 能精确分割目标 |
| **10px** | SAM 2 可接受范围，**主要评估指标** | SAM 2 能正确分割，边缘可能略有偏差 |
| **15px** | 宽松命中，仍能正常工作 | SAM 2 大部分情况能正确分割 |
| **20px** | 勉强可用，接近失败边界 | SAM 2 可能分割不准或包含背景 |

### 为什么选择 10px 作为主要指标？

1. **SAM 2 容错能力**：实验表明 SAM 2 在 10px 偏差内能正确初始化
2. **目标尺寸匹配**：小目标通常 30-50px，10px 约占 20-33%
3. **实际应用**：10px 在 1000x500 图像中是可接受的定位精度

## 距离统计

### 平均距离（Mean Distance）

**定义**：所有样本预测点与 GT 中心距离的平均值

```
Mean Distance = Σ distance_i / N
```

**特点**：
- 受异常值影响大
- 当前值：45.55px

**问题**：少数严重失败的样本会大幅拉高平均值

### 中位数距离（Median Distance）

**定义**：所有样本距离排序后的中位数

**特点**：
- 不受异常值影响
- 更能反映典型表现
- 当前值：7.81px

**分析**：
- 中位数 (7.81px) << 平均值 (45.55px)
- 说明：大部分样本表现良好，但存在少数严重失败的异常样本
- 改进方向：分析失败样本，针对性优化

### 距离分布分析

| 距离范围 | 样本比例 | 状态 |
|----------|----------|------|
| 0-5px | ~37% | 优秀 |
| 5-10px | ~22% | 良好 |
| 10-15px | ~9% | 可接受 |
| 15-20px | ~5% | 勉强 |
| > 20px | ~27% | 失败 |

## 置信度（Confidence）

### 定义

预测热力图峰值处的 sigmoid 输出值

```python
confidence = sigmoid(heatmap_logits[argmax_position])
```

### 物理意义

- 反映模型对预测的确信程度
- 范围：0-1
- 当前平均置信度：0.79

### 使用场景

1. **Re-prompting 触发**：置信度低于阈值时重新预测
2. **多目标选择**：选择置信度最高的目标
3. **结果过滤**：过滤低置信度的误检

## 指标优先级

1. **Hit Rate@10** - 主要指标，直接决定 SAM 2 初始化成功率
2. **中位数距离** - 辅助指标，反映典型定位精度
3. **Hit Rate@20** - 保守指标，反映整体可靠性
4. **置信度** - 辅助指标，用于 Re-prompting 判断
